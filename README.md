📌 Overview

This repository contains a collection of Java programs demonstrating essential core concepts. It is designed to strengthen understanding of fundamental Java topics through practical implementations.

🚀 Concepts Covered

🔹 Object-Oriented Programming (OOP)

🔹 Control Structures

🔹 Inheritance

🔹 Packages

🔹 Exception Handling

🔹 Multithreading

🔹 Strings

🔹 I/O Streams

🔹 Java Applications (Console-based)

🔹 Database Connectivity using JDBC

⚙️ Requirements
Java JDK 8 or higher

Any IDE (IntelliJ IDEA, Eclipse, VS Code) or terminal

Database (MySQL/PostgreSQL) for JDBC programs

▶️ Getting Started

1. Clone the Repository  
git clone https://github.com/your-username/java-core-concepts.git

2. Navigate to Project Directory
   cd java-core-concepts

3. Compile & Run a Program
   
javac src/folder_name/FileName.java
java src.folder_name.FileName
